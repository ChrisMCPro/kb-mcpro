# Ajouter des joueurs officiels FFF



1\. Dans le volet de gauche, cliquez sur le module **"JOUEURS"**

![](https://colony-recorder.s3.amazonaws.com/files/2023-08-08/a2948f76-5119-4003-99b5-e903caebade4/ascreenshot_cbd86f4f47934441a54da26efa392b66_text_export.jpeg)


2\. Depuis cette vue, cliquez sur le bouton **"Actions"**\
Choisissez **"Ajouter un joueur licencié"**

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-28/c3a5105a-d3b9-4b69-80f1-1627732deb56/stack_animation.webp)


3\. Recherchez vos joueur a partir de leur nom, leur numéro de licence ou leur année de naissance.

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-20/827535ef-de2a-4b2a-a33d-20bf5364ebdb/user_cropped_screenshot_a09ea243c2004e72812ddad06316715c_text_export.jpeg)


4\. Choisissez son poste.

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-20/7d0cf1c0-985e-4589-8083-c4ddf3f7866d/user_cropped_screenshot_55c258e3c8d24506a3b1977dc73b3e7e_text_export.jpeg)


5\. Puis cliquez sur **"ENREGISTRER"**, en bas de la liste pour terminer.

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-20/6e01dab4-5968-4e80-bb18-6525cacfa0bf/user_cropped_screenshot_f545d709569e4b68a2473fbe46cff8af_text_export.jpeg)
