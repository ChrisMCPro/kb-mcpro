# Éditer un joueur


1\. Cliquez sur le module **Joueurs**

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-02/ff0503f2-3ee3-4edd-a06c-16c735112120/user_cropped_screenshot_16a650a610654d9aa7225c56a12751dd_text_export.jpeg)


2\. Sélectionnez le joueur à modifier.

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-07/d8a91720-a39b-4040-b5b5-6a36064dfe50/ascreenshot_30b4afe59573448dbacec0c6184eabd8_text_export.jpeg)


3\. Cliquez sur le bouton **Actions** et choisissez l'option **Éditer**

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-02/7ae4f198-d57c-4db5-836c-130412091b6f/user_cropped_screenshot_8ce71c8e9e284d349cee77a012d15c71_text_export.jpeg)


4\. Modifiez l'adresse email.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-02/b515fcab-1188-4e61-9033-6efa13616c47/user_cropped_screenshot_19722e9fb49a41c7a2426ce83daaad61_text_export.jpeg)


5\. Cliquez sur **TERMINÉ** pour sauvegarder le changement.

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-07/a4c7959f-f51e-46da-aba7-691b16a0e7ac/ascreenshot_98e54bf8ac8f4f53b966eb31c7f7df1f_text_export.jpeg)
