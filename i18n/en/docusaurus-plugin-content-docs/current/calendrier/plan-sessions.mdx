# Planifier ses séances d'entraînement.


1\. **MyCoach Pro** vous donne la possibilité de gérer le calendrier de vos effectifs de manière complète, notamment en utilisant la fonctionnalité de planification d'événements.


2\. Cliquez sur le module **Calendrier**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-13/132f5eea-682f-436d-aebc-1f46d2e7e689/user_cropped_screenshot_55669cba0e5e4df1a9731b392610129c_text_export.jpeg)


3\. Pour commencer la planification d'événements 2 options sont possibles:

1. Cliquer sur une date et/ou un horaire et sélectionner l'option ***Planifier des séances***
2. Cliquer sur le bouton **+** et sélectionner l'option ***Planifier des séances***

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-13/3aebdc8f-5afe-43ea-8376-ad3fcbe5063a/user_cropped_screenshot_dd642eddce0443e094c8a2b05206004e_text_export.jpeg)


#### Récurrence simple


4\. Commencez la configuration de votre planification en sélectionnant:

1. **L'équipe concernée**
2. **Le type de convocation**
3. **La demande de présence**
4. **Les dates**

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-12/0bf7c9cb-047c-4b91-a406-58762edef31f/user_cropped_screenshot_5c3113d1cd2b433dbb660c06940ee4fd_text_export.jpeg)


5\. Laissez l'option **Récurrence** sur *Chaque semaine aux jours sélectionnés.*

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-12/f339f640-52d2-44ca-819e-12deb02b2f63/user_cropped_screenshot_20e9ee3161ee4ee88526db1934a5713b_text_export.jpeg)


6\. Paramétrez ensuite:

- Les jours pour lesquels planifier vos séances
- La séance elle même (type, thème, tags, horaire et durée, formulaire, infrastructures.)
- La visibilité aux joueurs

Puis cliquez sur **SUIVANT** pour voir le résumé de votre planification, ou **TERMINER** pour la valider.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-12/17bc5f2a-dd94-440d-afb2-a20f09dc2b05/screenshot_e19522ff5174417d92dfae80566e0bfb_text_export.jpeg)


#### Récurrence avancée


7\. La récurrence avancée vous permet de planifier plusieurs séances différentes par jour.\
Incluant une différenciation entre les tags, les type de séances ou encore les formulaires...


8\. À l'étape de choix de la **Récurrence**, choisissez l'option *Plusieurs événements récurrents différents.*\
\
Cliquez ensuite sur **SUIVANT**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-12/79ff2dd9-ee4d-4b27-9e30-91ad5d93411d/user_cropped_screenshot_10f8dc1652e8486d8458911cbe04b359_text_export.jpeg)


9\. Dans le même esprit que pour la récurrence simple, sélectionnez les jours pour lesquels vous souhaitez planifiés des séances.\
\
Vous pouvez alors les paramétrer indépendamment les unes des autres.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-12/c03977cc-bb40-474c-9c71-57d18c40512f/screenshot_6a9c688eb9f942a6b586a4a8b1fbc9d2_text_export.jpeg)


Tip: Si une journée se répète, un bouton vous permet de la copier, puis de la coller dans sur un autre jour.


10\. Si besoin paramétrer vos périodes ou vous excluez la planification.

Puis cliquez sur **SUIVANT** pour voir le résumé de votre planification, ou **TERMINER** pour la valider.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-12/ca04e6e7-2659-4321-984c-308d8b8bae58/screenshot_e378b0e1f67a4187bb9c480984058071_text_export.jpeg)
