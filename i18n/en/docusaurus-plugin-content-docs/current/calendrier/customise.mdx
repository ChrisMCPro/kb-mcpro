# Organisation et Visualisation


## Comment organiser votre calendrier?


Tip: Retrouvez également quelques conseils d'utilisations dans cette section


1\. Vous pouvez **déplacer un événement** via un **drag and drop**.

*Cela ne fonctionnera pas dans les cas suivants:*

- ***Les anniversaires et matchs officiels.***
- ***Les événements comportant une réservation d'infrastructure(s).***
- ***Vous n'êtes pas propriétaire de l'événement.***

![](https://colony-recorder.s3.amazonaws.com/files/2023-12-14/04bea8dc-28a0-49c2-b878-29c1e28f4c8a/ascreenshot_494bb3995b0645fc962804a5a8a4c38a_text_export.jpeg)


2\. *Si ce dernier a été planifié, il ne sera plus lié à la récurrence d'événements.*

![](https://colony-recorder.s3.amazonaws.com/files/2023-12-14/10455bcd-6d71-4d9a-bcf7-98386ccab22d/user_cropped_screenshot_564c4e9e4a15479ca38ff998030b7000_text_export.jpeg)


3\. Cliquer sur un événement vous permet d'afficher une **liste de raccourcis ou d'options**.\
(*Elle peut différer selon le type d'événement.*)

![](https://colony-recorder.s3.amazonaws.com/files/2023-12-14/ced01e04-ae90-4e84-9d87-42ef27bdd015/screenshot_57d0dad8e19441f8873b3ea89809393a_text_export.jpeg)


4\. Différentes vues sont disponibles pour votre planning:

- **Mensuel**
- **Hebdomadaire**
- **Jour**
- **Week-End**
- **Planning**
- **Liste**

![](https://colony-recorder.s3.amazonaws.com/files/2023-12-14/6ff39b8e-7d65-4543-9651-1199d71418bd/screenshot_fccf02f9757e415d92d6e11b0d0bde38_text_export.jpeg)


5\. Le calendrier **JOUEURS** vous permet de **consulter les événements auxquels un joueur est convoqué**, il est **disponible dans toutes les vues**, et vous **indique son statut de présence**.

Cliquez sur l'onglet JOUEURS, recherchez le joueur et cliquez sur son nom pour l'afficher.

![](https://colony-recorder.s3.amazonaws.com/files/2023-12-14/fda72761-c728-492a-876f-7f33d139598f/user_cropped_screenshot_fb9a4076f7224014955c94f6b72cf964_text_export.jpeg)


6\. Le calendrier **STAFFS** vous permet de **consulter les événements créés par un staff**, il est **disponible dans toutes les vues**.

Cliquez sur l'onglet JOUEURS, recherchez le staff et cliquez sur son nom pour l'afficher.

![](https://colony-recorder.s3.amazonaws.com/files/2023-12-14/7abac63d-f068-4415-8f40-979070ab5a8d/screenshot_7cae09ca734a4e31804096d55a464698_text_export.jpeg)


7\. Vous pouvez **utilisez les filtres pour optimiser la vue de votre calendrier**.

Cliquez sur le bouton dans le coin supérieur droit pour accéder aux filtres.

![](https://colony-recorder.s3.amazonaws.com/files/2023-12-14/460f583d-ff3e-412c-ad3c-9145122d5cea/ascreenshot_1fbf3cb422ef475686960b4cfc0b366c_text_export.jpeg)


8\. Vous pourrez **décocher les case pour masquer**:

- **Les types d'événements**
- **Les équipes**
- **Les types de séances**

Vous pouvez également choisir d'**afficher une couleur par type de séance, d'événement, ou une couleur par équipe**.

![](https://colony-recorder.s3.amazonaws.com/files/2023-12-14/fda4a682-db0b-4eca-b1df-266ff7be0c22/ascreenshot_325a3298b09945719aed2157d447bdb3_text_export.jpeg)
