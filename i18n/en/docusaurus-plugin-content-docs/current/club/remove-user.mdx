# Admin: Retirer tous ses droits à un utilisateur.
#### [Made by Support Team MyCoach Pro with Scribe](https://scribehow.com/shared/Admin_Retirer_tous_ses_droits__un_utilisateur__LQeouNqDSmWfH12PWptSvw)


1\. **ATTENTION:**\
\
Ce type d'action nécessite des droits administrateurs pour être accessible.\
Si vous n'y avez pas accès, renseignez vous auprès du détenteur du compte ou contactez notre service support à **support@mycoachpro.io** .


2\. Cliquez sur le module "**Club**"

![](https://colony-recorder.s3.amazonaws.com/files/2023-11-06/c3fa7ea9-680d-4981-8f54-8bc08c87eee0/ascreenshot_ed07db968d86468f9da6b817c64daa36_text_export.jpeg)


3\. Rendez-vous sur l'onglet "**STAFF**"

![](https://colony-recorder.s3.amazonaws.com/files/2023-11-06/6802c261-6c56-45ef-8979-44498b863670/ascreenshot_ea7a35ed3d5e4dc9888470a8a3e922f9_text_export.jpeg)


4\. Sur la ligne du staff cliquez sur la ***roue crantée,*** puis cliquez sur "**Retirer tous les droits**".\
Cette action a pour effet de retirer tous les droits de l'utilisateur sur toutes les équipes sur lesquelles il en avait.\
\
*Pour retirer des droits sur une équipe en particulier utiliser plutôt l'option "**Gérer les droits sur les équipes / groupes**".*
