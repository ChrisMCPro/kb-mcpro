# Admin: Remove all permissions from a user

1. **WARNING:**  
This action requires administrator rights to access.  
If you do not have access, ask the account owner or contact our support team at **support@mycoachpro.io**.

2. Click the **Club** module.

![](https://colony-recorder.s3.amazonaws.com/files/2023-11-06/c3fa7ea9-680d-4981-8f54-8bc08c87eee0/ascreenshot_ed07db968d86468f9da6b817c64daa36_text_export.jpeg)

3. Go to the **Staff** tab.

![](https://colony-recorder.s3.amazonaws.com/files/2023-11-06/6802c261-6c56-45ef-8979-44498b863670/ascreenshot_ea7a35ed3d5e4dc9888470a8a3e922f9_text_export.jpeg)

4. On the staff member’s row, click the ***gear icon,*** then click **Remove all permissions**.  
This action removes all permissions from the user on every team they had access to.

*To remove permissions for a specific team, use the **Manage permissions on teams / groups** option instead.*
