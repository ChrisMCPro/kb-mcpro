# Add values on nutrition properties

1. Click on the **Nutrition** module.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-26/888c536c-bf5f-4be7-adb4-c97c30369806/user_cropped_screenshot_a755c650bc18470c9bf9ae4350e1b655_text_export.jpeg)

2. You can enter a value for a test, for a player, **by using this field for today’s date.**

For a past date, **click the + button.**

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-26/0df8197c-149c-43c5-b057-5a16495506a7/File_fc1c68080287457f9ed3d36916849367_text_export.jpeg)

3. Choose **a date** and **enter a value**, then click **ADD**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-26/93825f83-25b6-438f-8b9f-a381071589dd/File_4cef9cab8b52473a90a8fbd52d73ad73_text_export.jpeg)

4. 

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-26/ca52b8de-257e-4638-9d30-27dabff52722/File_77aa0dc158e84e7f913c76db40c1307f_text_export.jpeg)

5. You can also add all tests performed for a player at once by clicking the **ADD VALUES** button.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-26/c331a044-cdef-4134-a8b1-6b1403a623ca/File_dcb3a51b88864cd2ab7a55ac176d4093_text_export.jpeg)

6. Choose a date.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-26/45873b46-2340-4fd2-8b85-3554622d4f33/File_e62e9868611040cb95ae7dc76b15ab6f_text_export.jpeg)

8. Enter the test values, then click **SAVE**.
