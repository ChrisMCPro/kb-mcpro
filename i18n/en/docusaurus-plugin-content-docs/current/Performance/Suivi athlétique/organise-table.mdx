# Ordonner les colonnes du tableau de suivi athlétique.
#### [Made by Support Team MyCoach Pro with Scribe](https://scribehow.com/shared/Ordonner_les_colonnes_du_tableau_de_suivi_athltique__K62am_IhSK68bDIBd_h7cQ)


1\. Pour modifier l'ordre des colonnes du tableau de suivi athlétique, il faut tout d'abord se rendre dans le module "**Suivi athlétique**"

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-11/71bca37d-dbb2-4257-837f-125f273872d6/ascreenshot_3ea5e1b98057438688a1f9d20ee9b166_text_export.jpeg)


2\. En **haut à droite** cliquez sur le **bouton des filtres**.

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-11/96adc429-a3ca-4a81-b526-f84817acf9e3/ascreenshot_562ee650e8234157bbcd62ef5a53ca6b_text_export.jpeg)


3\. Depuis ce panneau latéral, **vous pouvez modifier l'ordre** des colonnes **en cliquant sur les flèches haut et bas**.

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-11/b8fa15db-8b62-4881-970e-66ffe695ac60/ascreenshot_4463891fc7b641c0ac39cf5a8da65728_text_export.jpeg)


4\. **Les modifications seront enregistrée dans les préférences utilisateur.**\
**Chaque utilisateur peut donc avoir sa propre visualisation**.

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-11/eef18b5e-6b84-471a-82d3-e77ade690c64/ascreenshot_423e69002cda4ae0b372e0649dd704d9_text_export.jpeg)
