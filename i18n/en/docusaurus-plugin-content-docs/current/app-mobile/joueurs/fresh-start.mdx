# Joueur: démarrer avec l'application mobile
#### [Made by Support Team MyCoach Pro with Scribe](https://scribehow.com/shared/Joueur_dmarrer_avec_lapplication_mobile__jP0g1bIVSaq1E6R_u4BL8w)


1\. Le mail pour votre accès se présente comme suis et contient:

- **votre nom d'utilisateur** (il peut différer de votre email)
- **un lien pour créer votre mot de passe** **personnel** (valide 24h)

Cliquez sur le lien.

![](https://colony-recorder.s3.amazonaws.com/files/2024-07-18/792cf9e5-7392-491d-b5fb-0c19be3a3fd4/user_cropped_screenshot_2f851d8ec05d4914ac233aa74ef1df65_text_export.jpeg)


2\. Cette page vous permet de  créer votre mot de passe en respectant les discriminants indiqués:

- Saisissez votre mot de passe
- Confirmez votre mot de passe
- Cliquez sur **ENVOYER**
