# Create a player session from the mobile app. (player version)

#### Creating a player session

1. Tap the **Calendar** module to select a specific date **Create\*** from the *Day events* section.  
\**Go to step 8*

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/69e20313-a9c2-465e-b273-59bb50d5dbf9/File_cd2815388381423ea6f3f17fa73c9e1f_text_export.jpeg)

2.  
1. Choose the date on which you want to create your session.
2. Tap the event creation button.
3. Tap **Create a player session**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/15ff859d-66af-4bfb-a20a-4ad24caaf528/File_4be6d11a71de4ea4a0b19192e8bbb596_text_export.jpeg)

3. Then fill in your session details\*:

- Time
- Duration
- Theme
- Team
- Session type
- Form

Then tap **Done**.

\*If a staff member has provided you with a session template, tap the red import button, select the template to use, and continue the process.

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/d92131f1-a4ef-4bc1-b055-f125b91b95bf/File_83682709b3a74443bc5b4676c7c58e0e_text_export.jpeg)

4. Your session is created. Check the information and edit it if needed.  
If you have scheduled a form on this session, you will receive a notification when it is time to respond, just like for a session created by a coach.

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/d550f3ab-caa4-4586-897b-350b19acd35e/File_b083cfdf41904499808c19734d186456_text_export.jpeg)
