#### Création d'une séance joueur


1\. Tapez sur le module **Calendrier** pour sélectionner une date en particulier **Créer\*** depuis la section *Événements du jour*.\
\
*\*rendez-vous à l'étape 8*

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/69e20313-a9c2-465e-b273-59bb50d5dbf9/File_cd2815388381423ea6f3f17fa73c9e1f_text_export.jpeg)


2\. 1. Choisissez la date à laquelle vous souhaitez créer votre séance.
2. Tapez sur le bouton de création d'événement.
3. Tapez sur le bouton **Créer une séance joueur**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/15ff859d-66af-4bfb-a20a-4ad24caaf528/File_4be6d11a71de4ea4a0b19192e8bbb596_text_export.jpeg)


3\. Remplissez ensuite les informations de votre séance\*:

- Heure
- Durée
- Thème
- Équipe
- Type de séance
- Formulaire

tapez ensuite sur **Terminer**

*\*Si un staff vous a mis à disposition un template de séance, tapez sur le bouton d'import en rouge, sélectionner le template à utiliser et continuer le processus.*

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/d92131f1-a4ef-4bc1-b055-f125b91b95bf/File_83682709b3a74443bc5b4676c7c58e0e_text_export.jpeg)


4\. Votre séance est créée, vérifier les informations et modifiez si nécessaire.\
\
Si vous avez programmé un formulaire sur cette séance, vous recevrez une notification au moment d'y répondre comme pour une séance créé par un coach.

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/d550f3ab-caa4-4586-897b-350b19acd35e/File_b083cfdf41904499808c19734d186456_text_export.jpeg)
