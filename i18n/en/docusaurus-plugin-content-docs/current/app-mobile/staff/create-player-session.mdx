# Create a player session from the mobile app. (staff version)

#### Prerequisites

1. For a player to create their own sessions from their calendar in the MyCoach Pro - AMS mobile app, they must have the permission enabled.  
As staff, it is your responsibility to check that they have access.  
You can check their access from the player profile, in the **General information** tab, under **Player modules**.  
If they have access, a green checkmark appears next to the module.  
Be careful: module activation is done per team. If a player is on multiple teams, you must enable it on each one.

If you need help, contact us at [support@mycoachpro.io](https://mailto:support@mycoachpro.io)

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/1e894bed-d0c2-421f-a7ad-be1fec6bcc3c/user_cropped_screenshot_bb0d063adc524516b6703b8f56a821e3_text_export.jpeg)

2. If not, you must first grant this permission by editing the player profile. Click the **Action** button in the top right, then click **Edit**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/4b6cab16-ad3a-4c4b-a9b1-2a74d679c1bc/ascreenshot_abde372ea41b477ba6dc2526eddacf8b_text_export.jpeg)

3. At the bottom, you will find the Player modules section.

Select the module called **Player session**. This is what will allow your player to create their own sessions.

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/6b6d7cd3-25de-4465-9fcd-324f3072db4a/user_cropped_screenshot_8503db223e2c46e1b3d91387504d41c0_text_export.jpeg)

4. *Take the opportunity to review their information and correct it if needed.* Then click **Finish**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/c7cfdae2-b4ac-486d-9dc6-763b67386cf1/user_cropped_screenshot_e8422acd75ac47568ea5c3db19d9e118_text_export.jpeg)

5. If you want your player to answer the same form as usual for their own sessions, make sure that this form can be used on both sessions and player sessions.  
This is indicated by the **Staff and players** or **Players** label, depending on the selected option.

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/f32116c5-5eab-4f79-a7ea-7fc93c3c85fd/screenshot_2c9c29cacb094d4b9066e11e527157ae_text_export.jpeg)

#### Creating a player session

6. Tap the **Calendar** module to select a specific date **Create\*** from the *Day events* section.  
\**Go to step 8*

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/cd199d8a-a498-407f-b45a-9a6a6df61822/user_cropped_screenshot_b5433b3410e0455cbfb810f5ceefd429_text_export.jpeg)

7. 1. Choose the date on which you want to create your session.
2. Tap the event creation button.
3. Tap **Create a player session**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/36bdbdff-8fd5-4306-8a5c-0a0becd65f5f/screenshot_2a44003f236d4b9fa602ba239d5d17e8_text_export.jpeg)

8. Then fill in your session details\*:

- Time
- Duration
- Theme
- Team
- Session type
- Form

Then tap **Finish**.

\*If a staff member has provided you with a session template, tap the red import button, select the template to use, and continue the process.

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/9ee017d9-7eb9-435a-b87e-d6c2f6a5a04d/user_cropped_screenshot_a81fd0d6bb674fcd9f8505a348ba0f57_text_export.jpeg)

9. Your session is created. Check the information and edit it if needed.  
If you have scheduled a form on this session, you will receive a notification when it is time to respond, just like for a session created by a coach.

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/88766355-3cd2-4b35-bcb9-29a14c09e7fe/mobile_screenshot_ecbb7b5a13d24cb681d39696b030f55b_text_export.jpeg)
