# How do you manage form translations?

1. **MyCoach Pro** allows you to **manage the translations of your forms** so they can be adapted to as many languages as possible.

When creating a **form**, you can select a default language from the following: **French, English, German, Italian, Spanish, Dutch, and Portuguese**.

You can then fill in any missing translations (questions or answers) directly from your **form**.

**WARNING**

**Translations can only be edited on a form as long as it has not been used\***

\*used: scheduled on an event and has received responses

2. Go to **Insight > Forms > Configuration**
