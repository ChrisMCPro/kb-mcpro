# Create an RPE form from a template

#### [Made by Support Team MyCoach Pro with Scribe](https://scribehow.com/shared/Crer_un_formulaire_RPE_depuis_un_modle__6q591zLyS_yzpgUy_WbRiQ)

1. In the left side panel, click the **Insights** module.  
It will take you directly to the **Forms** sub-module.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/d8ba2a07-b0b2-4125-9dac-73255f3d0ff3/ascreenshot_31ae8c883886460abb8ef8a566698750_text_export.jpeg)

2. In the **Forms** sub-module, click the **CONFIGURATION** tab.  
This tab allows you to manage forms.  
*(Access to form configuration requires specific permissions and access rights.)*

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/3cf7fa47-8e22-4922-9c58-78f4a9e78b54/ascreenshot_de5761289eaa443d924d208f17fbe37b_text_export.jpeg)

3. Click the **ADD** button to start the creation process.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/fca58774-ca34-4bff-8a90-10f9118ba9d3/ascreenshot_bf88b8be942d47828d6e0106085cebc3_text_export.jpeg)

4. Click the **Form** field.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/76dfce0b-1090-4d69-a4c7-b7b5505b6cab/ascreenshot_cc8aade89e314cbf8616efe2854085f3_text_export.jpeg)

5. Click **Duplicate from template**, then select the form template you want to duplicate.  
For a post-session RPE form, this example uses the **Session (Post)** template.

*MyCoach Pro provides several pre-built form templates.  
They are a good starting point for using this feature, and you can edit them as needed.*

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/8bc37a61-67e5-4d91-a529-93062b4bf0f6/ascreenshot_f961c6e78aba400c9dc70dc430f50228_text_export.jpeg)

6. Choose the default language and the languages to import for translations.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/180b95be-ef77-4cca-bf68-05906e415316/ascreenshot_24cedf9fcfa6417192ed1930fb7e3b20_text_export.jpeg)

7. Click **Next**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/61e8aead-1987-46c0-aa3e-db3c4c142718/ascreenshot_7024610fbd154a82a657c2f50d7e5778_text_export.jpeg)

8. Configure the form as needed:

- **Name**: Choose the name of your form
- **Team**: Choose which team this form belongs to
- **Context**: Select the type of event on which this form will be used
- **Visible to players**: Make this form visible to players on an event
- **Type**: Choose how the form is displayed on mobile (single page or one question at a time)
- **Usable by**: Choose who can use this form (staff, player, or both)

Then click **FINISH**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/ee78406b-15eb-44eb-b80c-244400657062/user_cropped_screenshot_77b3a1123056436f80222c4578943810_text_export.jpeg)

9. Once the form has been created, you can configure it by clicking **CONFIGURE**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/b8215c6e-91ed-462f-886c-2b6f5c7c6e78/ascreenshot_89e16fbe2813470b83475887d4b1819f_text_export.jpeg)

10. For this post-session form, choose the **POST** phase.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/3a762757-1a04-4ea8-bccb-4f0b20d86027/ascreenshot_623b92fa159e4ac993a510454dfdc636_text_export.jpeg)

## Edit the form by adding a question

11. Click the **ACTIONS** button and choose **Generate a numeric question**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/01865a85-9704-4b0a-bfa3-52c7972c876f/ascreenshot_8d972cbffdda463aa2fc6680fd649bd3_text_export.jpeg)

12. Choose the type of numeric question you want to add.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/f5e0a32b-12e5-4d68-90e5-2e30b2cc1b90/ascreenshot_9398106841d3465490de1f3ad48a866e_text_export.jpeg)

13. Configure your question with the minimum answer, maximum answer, alert thresholds and color scheme.  
Then click **CREATE**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/aa6b9810-911b-432d-830a-5c78126cfbec/ascreenshot_0b896e5447e24fadb80254055ca47bd7_text_export.jpeg)

14. The new question is added to your form.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/1c7ba566-785d-4f91-a80f-98c9c07afd1f/ascreenshot_2ebc72a1e7e445d68330e4c83f6a3ce1_text_export.jpeg)

15. Select the new question and rename it as needed by entering a long label and a short label.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/a42f24ce-f45d-46ee-b0b1-07366e34ca67/ascreenshot_4a693aae8a484bbfa15891672b38cf72_text_export.jpeg)

16. Adjust the question settings if needed.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/26758360-a653-43d5-96f2-c201daa29da5/ascreenshot_5d1d9cec421040809699151e6681e657_text_export.jpeg)

17. Review the final display of the question in the form.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/c3442ba7-e0e1-4afb-b31b-c270a7606727/ascreenshot_356304be09ee4cb6baae23045df87cb9_text_export.jpeg)

18. Select the corresponding **analysis variable**.

*Analysis variables are an essential part of forms.  
They allow the data collected through your forms to be reported and analyzed in our **Report BI** tool.*

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/58626698-8dcb-47c0-9c19-bfde68ff131d/ascreenshot_e24c7394dd544073b15442b366463b7f_text_export.jpeg)

19. Confirm the selected analysis variable.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/70255d78-2940-4805-99e2-99cead30cee4/ascreenshot_358878bb275a41bfbf4a5c2207cdd8fe_text_export.jpeg)

20. Once your form is finished, click **Save and exit**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/2b3d8fad-4ff0-4a96-be01-68450aa78ef6/ascreenshot_2ab0b24d3d0345c7aacd32c3d48ff15c_text_export.jpeg)

21. If needed, you can then set this questionnaire as the default one so that it is automatically selected when creating a session.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/5f327da9-3525-4b6e-bb8f-8b1c380a755b/ascreenshot_7ecbab088a50493aabd912209655ae1c_text_export.jpeg)

22. Open the default form settings.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/1fd63286-02bf-4e3c-b755-c114bcc38be3/ascreenshot_f0b51657c7e64ed8a5e93a5e74d283ea_text_export.jpeg)

23. Confirm the form as the default questionnaire.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/a97bf055-771d-4ef6-b4ae-bf8a142446a3/ascreenshot_ff2971bcfe39428ea19209c936350443_text_export.jpeg)
