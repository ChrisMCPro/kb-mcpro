# Create a report and share it with your players

1. Report creation can be done from several places in **MyCoach Pro**.  
In this example, we start from **Home**, in the **REPORT BI** tab.

2. Click the Report BI dashboard from which you want to create a report.

![](https://colony-recorder.s3.amazonaws.com/files/2024-09-09/5d7667bf-2072-4b70-ab9c-5398ea172dea/ascreenshot_60415bd4c4a94735a456b7eaa9a985a4_text_export.jpeg)

3. Once the dashboard has loaded with the desired data, click **CREATE A REPORT**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-09-09/d24d49e8-99b0-4e87-be37-f6f41c5565a1/ascreenshot_363239806f984c84b000621c6a560bb3_text_export.jpeg)

4. You can assign a status to your report if you have created one, and edit its title. *(Optional)*

![](https://colony-recorder.s3.amazonaws.com/files/2024-09-09/6a9c5eb2-9a2f-4061-a9f8-03e1cf2ddcc7/ascreenshot_cc3284b4f57045b3ae98da6eb9d508f8_text_export.jpeg)

5. Click this button to generate 1 report per player who has data on this dashboard.

![](https://colony-recorder.s3.amazonaws.com/files/2024-09-09/2e4988f1-bdb1-426b-b8d5-4ed849d94b32/ascreenshot_183ff42dfd5a4532b5e131195a6861d0_text_export.jpeg)

6. Click this button to send the report to your players who have data on this dashboard.  

**WARNING:** If you choose this option without selecting ***One report per player***, players will be able to see other players’ data.

![](https://colony-recorder.s3.amazonaws.com/files/2024-09-09/2bdda778-9d9e-4156-ae76-2022d9eff975/ascreenshot_e414f187077844298c850c38ee774e0c_text_export.jpeg)

7. Click **Next** twice.

![](https://colony-recorder.s3.amazonaws.com/files/2024-09-09/f74d8db3-f8b6-4a36-88ad-f2e0fbb04b7b/ascreenshot_3e910fcce1f3424ea1f465e2e79bd7f3_text_export.jpeg)

8. Click **DONE**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-09-09/9e10b181-ad21-487f-8a94-462856ebe3cf/ascreenshot_343edee28a9f4c869f33280a570db56b_text_export.jpeg)
