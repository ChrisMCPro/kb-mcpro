---
title: "Add a medication"
description: "Create a medication database for prescriptions."
---

1. In the *module panel*, click the **Medical** module.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/c2443ea2-b6cd-4ca7-8924-3e08e99e5c24/ascreenshot_77af2196acb449cfb17ea6c0bbe8af30_text_export.jpeg)

2. Select the **Medications** tab.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/9beea380-3025-49e4-b7b8-823fc2cd7f3b/ascreenshot_1d1bdeb1bacc460f93344db2d9429a13_text_export.jpeg)

3. Click **ADD A MEDICATION**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/6576beaf-ca80-4791-a43e-a997d1715445/ascreenshot_92d5d5400bd445b18310538f5e9edec0_text_export.jpeg)

4. Fill in at least the required fields to create the medication:

- **Name:** the medication name
- **Category:** define the medication type (*medication / supplement / other*)
- **Visibility:**
  - Select the roles that can use this medication in a prescription, among: *Medical / Paramedical / Nutrition*

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/45185d7e-3487-43fe-b8ee-83b4548b1a7c/screenshot_384cbc3e769f4f9892ce814440609bcc_text_export.jpeg)
