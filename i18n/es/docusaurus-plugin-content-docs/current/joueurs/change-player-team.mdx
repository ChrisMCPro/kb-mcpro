# Changer un joueur d'équipe.


1\. Depuis le module **Joueurs**, choisissez le joueur que vous souhaitez changer d'équipe.

![](https://colony-recorder.s3.amazonaws.com/files/2024-09-17/1082d587-3ed5-4957-9f03-201e4b420fba/ascreenshot_21754bc787da43b18dbb135a3387539a_text_export.jpeg)


2\. 

![](https://colony-recorder.s3.amazonaws.com/files/2024-09-17/eb3ed081-8f3e-41d6-bd7b-52e598a6cdf6/ascreenshot_f0787106b8e546f7b343219e90e4c746_text_export.jpeg)


3\. Cliquez sur le bouton **Actions**, et choisissez l'option **Changer d'équipe**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-09-17/74568885-bf5c-4d4c-a688-94e835d4bebf/ascreenshot_65846939762f4c0788e87c3eedbd35a1_text_export.jpeg)


4\. 

![](https://colony-recorder.s3.amazonaws.com/files/2024-09-17/9fc179b3-0da1-4df1-ae60-5302b92cee98/ascreenshot_ee776903c03943fbb60892a4ad1e01e3_text_export.jpeg)


5\. **Choisissez les équipes source et de destination**, si vous n'avez accès qu'à deux équipes, le choix sera figé.\
**Cochez la checkbox** si le joueur doit conserver des données de son passage dans l'ancienne équipe.

![](https://colony-recorder.s3.amazonaws.com/files/2024-09-17/0d219405-6507-45d2-91b7-0c7ed6819008/user_cropped_screenshot_d9af6020df494ede84553062bf688e47_text_export.jpeg)


6\. Cliquez sur **TERMINER**

![](https://colony-recorder.s3.amazonaws.com/files/2024-09-17/37fdc20e-f072-432d-aa1e-569e61419a3b/File_8fef569b2e9c47dbbbfefcc786199bcf_text_export.jpeg)
