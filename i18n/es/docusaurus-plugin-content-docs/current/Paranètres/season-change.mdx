# Effectuer le passage de saison
#### [Made by Support Team MyCoach Pro with Scribe](https://scribehow.com/shared/Effectuer_le_passage_de_saison__ixBXDUgJT6GQnE7ZjZypmA)


1\. Rendez-vous dans **Paramètres &gt; Changement de saison**


2\. Cherchez **les joueurs** dans les **équipes des saisons précédentes**, et cliquer sur le bouton **\+** pour les **basculer sur la saison en cours.**

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-19/50cb8be4-8eed-4026-ac27-ce42b0a3a75f/ascreenshot_ab27b377c3d541f98e7877fa358c7a7b_text_export.jpeg)


3\. Cliquez sur "**SAUVEGARDER**"

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-19/e8aec7ee-70be-49a2-8ffa-731bacbd030b/ascreenshot_39f2c1fae47e4e19a4ee65661ff925b0_text_export.jpeg)


4\. Vous avez maintenant accès à la nouvelle saison. Cliquez sur la saison dans la barre de navigation.

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-19/1a43abe4-eab2-4e08-8990-49b15c163c9a/ascreenshot_7ab0686c3cb444f2a345a8ba4522b2c5_text_export.jpeg)


5\. Choisissez 2023/2024

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-19/6ca89567-0d6f-4a4a-830c-f5e0ee9f3c9d/ascreenshot_55c6ecb23b1341a7a95bbe4fe677c880_text_export.jpeg)


6\. Vérifiez la présence de vos joueurs dans votre effectifs en cliquant sur le module "**JOUEURS**"

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-19/7895cd5e-179d-4795-b62b-2b620e4c4cef/ascreenshot_4965264620fd4da0955adfb52977869e_text_export.jpeg)
