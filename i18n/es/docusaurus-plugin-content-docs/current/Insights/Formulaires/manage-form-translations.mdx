# Gestionar las traducciones de los formularios

1. **MyCoach Pro** te permite **gestionar las traducciones de tus formularios** para adaptarlos a la mayor cantidad de idiomas posible.\
\
Al crear un **formulario**, puedes seleccionar un idioma por defecto entre los siguientes: **francés, inglés, alemán, italiano, español, neerlandés y portugués**.\
\
Después podrás completar las traducciones que falten (preguntas o respuestas) desde tu **formulario**.

**ATENCIÓN**

**Las traducciones solo se pueden modificar en un formulario mientras este no haya sido utilizado\***\
\
*\*utilizado: programado en un evento y con respuestas recibidas*
