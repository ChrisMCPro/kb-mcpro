# Consultar las respuestas de un formulario desde el evento

1. Ve a tu **Calendario** haciendo clic en él.

![](https://colony-recorder.s3.amazonaws.com/files/2023-11-13/ae38f903-86ce-4288-a87f-68a4b83936ce/user_cropped_screenshot_3c5d5376e7fb42e8b786c9157ad6d35c_text_export.jpeg)

2. Para consultar las respuestas de un cuestionario en un evento de seguimiento, haz clic en él y selecciona **Ver**.

![](https://colony-recorder.s3.amazonaws.com/files/2023-11-13/077d7c00-1f81-4f38-a9fb-9656766b3028/stack_animation.webp)

3. Para consultar las respuestas de un cuestionario de sesión o de partido, haz clic en él y selecciona **Cuestionario**.

![](https://colony-recorder.s3.amazonaws.com/files/2023-11-13/4431649a-13b8-4f2f-a0ac-13707c11ddf4/stack_animation.webp)
