# Dupliquer un formulaire
#### [Made by Support Team MyCoach Pro with Scribe](https://scribehow.com/shared/Dupliquer_un_formulaire__V29iPXrjS2yumAfZii4YAA)


1\. Cliquez sur "**Insights**"

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-08/daa01a05-8adf-4874-9302-6baf691319ad/ascreenshot_afdb8104cbf2404381bc3d248db26fbd_text_export.jpeg)


2\. Sélectionnez l'onglet "**CONFIGURATION**"

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-08/fa99c4fc-895c-47ae-a1b2-76aae9c70c70/ascreenshot_34b2f215f6464f1fb47d23b6ae3a2cef_text_export.jpeg)


3\. Devant le questionnaire que vous souhaitez dupliquer, cliquez sur la roue.

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-08/a9266817-f486-4ea1-ac34-fc96e19afdd2/ascreenshot_ce2f876deeee4c3dafe03d9bf75d39a7_text_export.jpeg)


4\. Cliquez sur "**Dupliquer**"

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-08/f3b275d7-c2d2-4e84-9011-0be1fe7c1174/ascreenshot_6fbc7f52d8394cf3a73f70bc69659f49_text_export.jpeg)


5\. Configurez votre questionnaire

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-08/ccec4f85-f4d4-42c5-bfbe-4227e7ec54d2/ascreenshot_7116f9d0636f4b51a5b25413d49d81fa_text_export.jpeg)


6\. 

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-08/a857b1b1-63fb-48f7-bc9c-9df7e515d598/ascreenshot_2df073345eaa4285926622c094878b4c_text_export.jpeg)


7\. Sélectionnez l'équipe sur laquelle vous souhaitez dupliquer ce questionnaire.\
(Pour la voir elle doit être sélectionnée parmi vos **Équipes actives**).

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-08/e2c316e8-9224-4710-ae03-25cf3d89289f/ascreenshot_be0b67bae41d4001885d9d3f07422693_text_export.jpeg)


8\. Une fois la configuration terminée, cliquez "**TERMINÉ**"

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-08/f5db754d-4c31-492c-8a7e-77870159a6fd/ascreenshot_c7fb41725909437188ec36285fe663f2_text_export.jpeg)


9\. Vous devez maintenant activer votre questionnaire.\
Cliquez sur "**ACTIONS**" puis "**Activer**"

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-08/3628c255-8049-4e08-9914-fa8264e6f444/ascreenshot_9dd0ddc793ed44e18f8b9f05b8696ea6_text_export.jpeg)


10\. 

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-08/fc6a261d-6516-4d63-b790-475f34dbec2f/ascreenshot_aa53c1e8d57c4f5b8254ee6363d85efa_text_export.jpeg)


11\. Choisissez si vous souhaitez ou non activer ce questionnaire par défaut, puis cliquez sur "**ACTIVER**" pour terminer.\
Le questionnaire est maintenant disponible.

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-08/659e36e5-1a58-4227-9290-6337536ddd92/ascreenshot_4a7e8b4d29b64d43871e39508e6c5ea2_text_export.jpeg)
