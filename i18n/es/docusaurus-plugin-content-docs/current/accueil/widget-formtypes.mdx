# Widget: États de forme


#### Ajouter le widget des États de forme


1\. Depuis l'[Accueil](https://app.mycoachpro.io/dashboard) cliquez sur le bouton **Filtres**

![](https://colony-recorder.s3.amazonaws.com/files/2025-01-16/d16e95fc-dd0f-40fd-a5b0-852bf609bb8f/ascreenshot_487844ad52174c82a7cdba93b3aa56e2_text_export.jpeg)


2\. Pour afficher le widget cochez la case *Voir les états de formes*.

![](https://colony-recorder.s3.amazonaws.com/files/2025-01-16/49a0bd5d-43ea-48ee-bbab-58fd30c0c115/ascreenshot_79288728b1b84eadaa37480a005444d0_text_export.jpeg)


3\. Sélectionnez l'équipe à afficher si vous en avez plusieurs.

![](https://colony-recorder.s3.amazonaws.com/files/2025-01-16/d0671a61-3135-4320-b716-cf7dbc01e9d0/user_cropped_screenshot_d47fc7f2b54342a384e84f79a67541d9_text_export.jpeg)
