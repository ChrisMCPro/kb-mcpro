# Crear un ejercicio

1. Haz clic en el módulo **Ejercicio**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-12/a475c91b-90e1-4217-b8e2-47acd3b4cd7a/screenshot_2ae9108c0fe046e89f810da6ef0c36c8_text_export.jpeg)

2. Haz clic en el botón **Añadir un ejercicio**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-12/ad9839d9-9fdd-4378-9e7e-6b218f018ce6/screenshot_826ee0bd34f44cb8a72638e69d9b9de1_text_export.jpeg)

3. Luego selecciona **Añadir un ejercicio del club**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-02-12/a16c4a52-8dc9-46c3-b92b-f3af2e24f9f4/ascreenshot_1910313fa9534cc897a338244d232a46_text_export.jpeg)

4. Completa tu ejercicio rellenando los campos de información y descripción.\
Esta información será útil para facilitar su búsqueda o cuantificar su uso.

![](https://colony-recorder.s3.amazonaws.com/files/2024-02-12/d0b6951e-eed2-4699-bc4c-e62b805db104/ascreenshot_db9dd8722fee4c0ca32cd5af31821825_text_export.jpeg)

5. Si es necesario, añade **parámetros dinámicos** por defecto. Son muy útiles para indicar con precisión los criterios de un ejercicio, como el número de repeticiones, el peso, la intensidad, etc.\
\
*Se pueden modificar al añadir un ejercicio en una sesión.*\
\
Luego haz clic en **SIGUIENTE**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-02-12/cdf40a46-431d-40bd-84c8-c6f87bf754ca/ascreenshot_81aa7217862240a897a3f00decb05e7e_text_export.jpeg)

6. Haz clic en **CREAR** para utilizar nuestro editor visual, o en **SUBIR** para importar tus propios elementos visuales, PDF o vídeos.

![](https://colony-recorder.s3.amazonaws.com/files/2024-02-12/828853d3-47c6-4de6-a675-dfa898e6fc7c/ascreenshot_d772824bee30414695a6784dc83fc892_text_export.jpeg)

7. El editor visual te permite utilizar numerosos tipos de campos, materiales o atletas.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-12/610724bb-08b7-4a12-819d-ad91a901d6f0/screenshot_e948397ebbd0468eb08e1f298af13f1b_text_export.jpeg)

8. Haz clic en **AÑADIR EL VISUAL** para incorporarlo al ejercicio.\
\
*Es posible añadir varios para cada ejercicio.*

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-12/f46985f9-ff56-4f7d-90a3-cd2dd03b6baf/user_cropped_screenshot_50eae3c7d1c6448f9022498dcb125e84_text_export.jpeg)

9. Haz clic en **TERMINAR** para finalizar la creación de tu ejercicio.

![](https://colony-recorder.s3.amazonaws.com/files/2024-02-12/b14be6af-6a3c-48db-a1db-3611cd51563e/ascreenshot_e04774cb8570499d960f860f67ed8f2b_text_export.jpeg)
