# Synchroniser des données de nutrition à l'aide d'un connecteur.
#### [Made by Support Team MyCoach Pro with Scribe](https://scribehow.com/shared/Synchroniser_des_donnes_de_nutrition__laide_dun_connecteur__yY8JuO7fSkGEk9WQh9nCiA)


#### Synchroniser ses données athlétiques depuis un connecteur.


1\. Depuis le volet de gauche, cliquez sur le module **Performance** &gt; **Suivi Athlétique**

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-26/a999f064-33cc-49cc-bda9-0c5086261cf6/File_1da05115aa024107aed12e5244da503d_text_export.jpeg)


2\. Pour synchroniser les données de suivi athlétique via un connecteur créé en amont, cliquez sur le bouton **SYNCHRONISER DES VALEURS**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-26/25bcb3b2-5136-4f6a-8a7b-0f9f2b8cccb1/File_5f16ee9e67764171ac52bd6e6f4032aa_text_export.jpeg)


3\. Choisissez la **date d'importation**, cela vous permet de synchroniser des données collectées à une date antérieure.\
Puis **faites glisser votre fichier .csv** dans la zone indiquée, ou cliquez pour aller le sélectionner sur votre appareil.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-26/14dec7b6-5c37-4069-b73a-7b06121a0092/File_b34bc34ec09f4a298df749a30075cb37_text_export.jpeg)


4\. Choisissez le connecteur à utiliser en cliquant sur **SÉLECTIONNER**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-26/1dfdfc48-7133-4e73-934f-c54b57a06979/File_87869f0cd29b4fadad9d2f53ceaced74_text_export.jpeg)


5\. Vérifiez la correspondance entre les joueurs de votre fichier et ceux de MyCoach Pro.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-26/94f7b656-8f77-453f-88b7-01fca3882dd1/File_0ad1f91c03b5433b8ca6bb2cd3cbaaca_text_export.jpeg)


6\. Pour terminer l'import de vos données, cliquez sur **IMPORTER**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-26/a01bea5a-9657-4199-8524-5090ce11f8e7/File_b788fe7323054013bdfbd34fb970ee7e_text_export.jpeg)
