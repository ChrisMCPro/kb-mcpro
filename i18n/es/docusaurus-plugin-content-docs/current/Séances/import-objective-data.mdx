# Importar datos objetivos en una sesión

1. Desde tu calendario, haz clic en la sesión para mostrar los accesos directos.

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-05/4b7c356a-4f6d-4be9-a1f9-85d7cbdbf886/ascreenshot_e237389154de4391a2eb3bbdff65d1d7_text_export.jpeg)

2. Selecciona **GPS** para ir directamente a la pestaña **Datos objetivos** de la sesión.

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-05/2c039684-16a6-489a-bce5-4edb8fb9576b/ascreenshot_dffd190b431e47d4bef65f2ff4d01a64_text_export.jpeg)

3. Haz clic en el botón **SINCRONIZAR UNA SESIÓN**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-09-19/067e4b2b-5d26-4cca-ab2c-98624d6bba60/user_cropped_screenshot_9af8fe740dc6488c9c0fe554780642b7_text_export.jpeg)

4. Selecciona el conector que deseas utilizar.

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-05/fe17e589-8678-4753-8ab2-9f592d89237c/ascreenshot_9520b766bd7842e8961e7cfc8b14142b_text_export.jpeg)

5. Arrastra y suelta tu archivo en la zona\
o\
Haz clic en la zona punteada para buscar y seleccionar el archivo necesario para importar tus datos.

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-05/eece73c9-9ae6-4ca8-bc72-fbd4d19094f4/ascreenshot_e70260a09a9149ad919579910e710206_text_export.jpeg)

6. Haz clic en **Importar** para validar la importación de los datos objetivos en tu sesión.

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-05/8483ba5b-efc2-439b-be61-108b2e8ea4b6/ascreenshot_4a70232ef76144ae808dd28636a62c95_text_export.jpeg)
