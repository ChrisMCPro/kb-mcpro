# Ajouter une valeur sur un test athlétique
#### [Made by Support Team MyCoach Pro with Scribe](https://scribehow.com/shared/Ajouter_une_valeur_sur_un_test_athltique___vaAbw4WTdGhMjfAxfqwFA)


1\. Cliquez sur le module **Performance &gt; Suivi Athlétique**

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-31/c12fb78f-9984-44c0-ab6c-45f258c0451b/ascreenshot_da2abfd445974f0fbfc9e64e71a75dc5_text_export.jpeg)


2\. 

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-31/68a3b3b3-e687-4901-875b-77335c96d690/ascreenshot_f0706570d3f64edd869a7f6ce56a21c8_text_export.jpeg)


3\. Vous pouvez entrer une valeur sur un test pour un joueur en utilisant le champs de texte pour la date du jour.

Pour une date antérieure, cliquez sur le bouton **\+** .

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-31/27d45e32-42dd-4113-8ef6-efd484e70acc/ascreenshot_4106ba85f294463abf4e5dcc24edc039_text_export.jpeg)


4\. Choisissez **une date** et **entrez une valeur**, puis cliquez sur **AJOUTER**

![](https://colony-recorder.s3.amazonaws.com/files/2023-11-29/24b90e26-2def-4546-9a31-696057810bea/screenshot_4c2d3e6ab3d6476eaf6f67df6f1c35e7_text_export.jpeg)


5\. 

![](https://colony-recorder.s3.amazonaws.com/files/2023-11-29/b578b926-d7cd-44e7-9c18-e56b5e11dda1/screenshot_c7dd226ad9ec4644aeaa4de730ad8277_text_export.jpeg)


6\. Vous pouvez également ajouter pour un joueur tous les tests effectués en une seule fois en cliquant sur le bouton **AJOUTER DES VALEURS**.

![](https://colony-recorder.s3.amazonaws.com/files/2023-11-29/3f5a2194-d010-4124-b78a-0468ad1a6f95/user_cropped_screenshot_a1014b49eb344fb2b771d2eaafcfcd46_text_export.jpeg)


7\. Choisissez une date.

![](https://colony-recorder.s3.amazonaws.com/files/2023-11-29/e755b894-e0e1-485e-9e54-d3164f8d1fcd/user_cropped_screenshot_3c291ca62a3d451299ce4e1d03c6b73c_text_export.jpeg)


8\. 

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-31/ae016304-7ce7-457d-8e0f-6858db105fbe/ascreenshot_c7d5ae3ab5fe4a408e296e9f08e7ed2a_text_export.jpeg)


9\. Entrez la valeur des tests, puis cliquez sur **ENREGISTRER**.

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-31/6045d3b8-07fa-4995-aee0-e61a8069cc95/ascreenshot_206d4f3efd1c495294b0adc813b298e7_text_export.jpeg)
