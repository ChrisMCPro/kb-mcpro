# Sincronizar datos de seguimiento atlético mediante un conector

#### Sincronizar sus datos atléticos desde un conector.

1. Desde el panel izquierdo, haz clic en el módulo **Rendimiento** > **Seguimiento atlético**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/099ee12b-d0d6-4642-a5bb-353a7b329dc0/ascreenshot_eab4f77fe9e44bb09e48fce99666bbb0_text_export.jpeg)

2. Para sincronizar los datos de seguimiento atlético mediante un conector creado previamente, haz clic en el botón **SINCRONIZAR VALORES**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/dd5c8b9f-46e5-4740-bbae-d6b226e73d96/ascreenshot_eb8e95dfbc424099869cfc0d5827a2d1_text_export.jpeg)

3. Elige la **fecha de importación**; esto te permite sincronizar datos recogidos en una fecha anterior.  
Luego **arrastra tu archivo .csv** al área indicada, o haz clic para seleccionarlo desde tu dispositivo.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/7c9d44f1-e3f7-4f4f-96ec-e709f7a7ca8d/user_cropped_screenshot_df6b7a147dec4c8ca9c06e0abfcca7e7_text_export.jpeg)

4. Elige el conector que quieres usar haciendo clic en **SELECCIONAR**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/53a7e3ae-c590-45a6-a001-ab2ca8a0d960/user_cropped_screenshot_6d726e37eb21406cacf32b0994441b5a_text_export.jpeg)

5. Verifica la correspondencia entre los jugadores de tu archivo y los de MyCoach Pro.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/2fa6f411-bb44-4604-ab74-cb3974fbd637/ascreenshot_8ffb14700e4c4d8ebada692ec524aa32_text_export.jpeg)

6. Para finalizar la importación de tus datos, haz clic en **IMPORTAR**.

![](https://colony-recorder.s3.amazonaws.com/files/2024-11-26/33f60d1c-c99b-4887-ab83-902bb0ca2c05/ascreenshot_87b3f411d0c749f482912f3ff7766a2a_text_export.jpeg)
