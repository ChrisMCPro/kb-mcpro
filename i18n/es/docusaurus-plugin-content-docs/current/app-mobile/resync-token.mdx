# Resynchroniser son token de notification
#### [Made by Support Team MyCoach Pro with Scribe](https://scribehow.com/shared/Resynchroniser_son_token_de_notification__ay6pVuuNSWm8uSlIM31P6Q)


1\. Connectez-vous sur votre compte sur l'application mobile **MyCoach Pro - AMS**, et cliquez sur le **Menu.**

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-25/1606c72e-d5da-43e1-afda-cf2da863bd6b/user_cropped_screenshot_afbc6b3966694f3e8d6c79b1339abf2a_text_export.jpeg)


2\. Dans ce nouveau menu, défilez vers le bas, puis dans la section **Autre**, sélectionnez **Diagnostic**

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-25/55098f12-cff6-4a91-872c-dfbbb6873396/user_cropped_screenshot_01e7f9cc8b78422ba10e6e15d6a6fe17_text_export.jpeg)


3\. Enfin, cliquez sur **Resynchroniser le token de notification**.

![](https://colony-recorder.s3.amazonaws.com/files/2023-09-25/45a5d174-e281-4ae3-a266-aad3a5fb85e9/user_cropped_screenshot_26e06701b3134bffa93f91442db09823_text_export.jpeg)
