# Crear una sesión de jugador desde la aplicación móvil (versión jugadores)

## Crear una sesión de jugador

1. Toca el módulo **Calendario** para seleccionar una fecha concreta y luego toca **Crear\*** desde la sección *Eventos del día*.  
   \* *Ve al paso 8.*

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/69e20313-a9c2-465e-b273-59bb50d5dbf9/File_cd2815388381423ea6f3f17fa73c9e1f_text_export.jpeg)

2. Sigue estos pasos:
   1. Elige la fecha en la que quieres crear tu sesión.
   2. Toca el botón de creación de evento.
   3. Toca **Crear una sesión de jugador**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/15ff859d-66af-4bfb-a20a-4ad24caaf528/File_4be6d11a71de4ea4a0b19192e8bbb596_text_export.jpeg)

3. Completa la información de tu sesión\*:
   - Hora
   - Duración
   - Tema
   - Equipo
   - Tipo de sesión
   - Formulario

   Luego toca **Finalizar**.

   \* *Si un miembro del staff ha puesto a tu disposición una plantilla de sesión, toca el botón de importación en rojo y selecciónala.*

![](https://colony-recorder.s3.amazonaws.com/files/2025-06-10/6e4f9f30-6ed0-4fb4-95ff-cc1a55b957c1/File_ec63efb2d2dc48a1839b73cf4ba97d3f_text_export.jpeg)
