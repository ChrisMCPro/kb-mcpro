🔄 Sincronización con Apple Salud o Health Connect

Antes de continuar, asegúrate de que tu aplicación de seguimiento de actividad autoriza el acceso a sus datos desde **Apple Salud** (iOS) o **Health Connect** (Android).

A continuación encontrarás las documentaciones oficiales de las aplicaciones de seguimiento sobre la sincronización:

| Aplicación | Documentación / sincronización |
|---|---|
| Garmin Connect |  |
| Polar Flow |  |
| Whoop |  |

La primera vez que abras la aplicación, se te pedirá sincronizar tus datos de salud:
- iOS: **Apple Salud**
- Android: **Health Connect**

✅ Acepta **todos** los permisos solicitados: solo se utilizarán los datos relativos a **distancia**, **velocidad** y **frecuencia cardíaca**.

Esto permite que la aplicación importe automáticamente tus sesiones de entrenamiento registradas (desde Polar, Garmin, etc.).

- Algunas aplicaciones de seguimiento de actividad necesitan aplicaciones de terceros para establecer la sincronización con Google Fit y Health Connect. Es el caso, por ejemplo, de **Huawei Salud**, que requiere el uso de **Health Sync**.
