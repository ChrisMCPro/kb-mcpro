# Jugador: empezar con la aplicación móvil

1. El email de acceso se presenta así e incluye:
   - **tu nombre de usuario** (puede ser distinto de tu email)
   - **un enlace para crear tu contraseña personal** (válido 24h)

   Haz clic en el enlace.
