# Planifier ses séances d'entraînement
#### [Made by Support Team MyCoach Pro with Scribe](https://scribehow.com/shared/Planifier_ses_sances_dentranement__2giKa2N8SwmYUfrUXlr7NQ)


1\. Cliquez sur le module **Séances.**

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-13/fa40bd77-18d5-4890-8625-d9722e72b791/user_cropped_screenshot_3db2e7918b8c465ba91a4e6652e54b0d_text_export.jpeg)


2\. Pour commencer la planification cliquez sur le bouton **Planifier**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-13/b3a3ffb4-a9f8-4c93-8208-df9f5827ea68/user_cropped_screenshot_5b4b863a3a3a4d799a2bfb71720c9303_text_export.jpeg)


#### Récurrence simple


3\. Commencez la configuration de votre planification en sélectionnant:

1. **L'équipe concernée**
2. **Le type de convocation**
3. **La demande de présence**
4. **Les dates**

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-13/1a7e52f0-e2ab-4c04-af61-25dab27bfaf6/File_7bb41257185246de97cc3363ca3bb843_text_export.jpeg)


4\. Laissez l'option **Récurrence** sur *Chaque semaine aux jours sélectionnés.*

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-13/d179beee-5cd4-492a-a400-df7b699bef99/File_cbb726d8193144459c1a7d1d2e4d923f_text_export.jpeg)


5\. Paramétrez ensuite:

- Les jours pour lesquels planifier vos séances
- La séance elle même (type, thème, tags, horaire et durée, formulaire, infrastructures.)
- La visibilité aux joueurs

Puis cliquez sur **SUIVANT** pour voir le résumé de votre planification, ou **TERMINER** pour la valider.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-13/74d1246b-a690-4751-a924-953d7a83a5e2/File_74ea30e7317f46379bc0aeaf69853c2a_text_export.jpeg)


#### Récurrence avancée


6\. La récurrence avancée vous permet de planifier plusieurs séances différentes par jour.\
Incluant une différenciation entre les tags, les type de séances ou encore les formulaires...


7\. À l'étape de choix de la **Récurrence**, choisissez l'option *Plusieurs événements récurrents différents.*\
\
Cliquez ensuite sur **SUIVANT**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-13/310a0f9a-2ef5-4656-bf2e-9a85aa51d372/File_db249d2f5773494bb1649a434ad2fcfb_text_export.jpeg)


8\. Dans le même esprit que pour la récurrence simple, sélectionnez les jours pour lesquels vous souhaitez planifiés des séances.\
\
Vous pouvez alors les paramétrer indépendamment les unes des autres.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-13/18a803a4-f75f-43a1-b4cd-bc66a7f115e2/File_09c1546154ba40bc9816776fce9de12b_text_export.jpeg)


Tip: Si une journée se répète, un bouton vous permet de la copier, puis de la coller dans sur un autre jour.


9\. Si besoin paramétrer vos périodes ou vous excluez la planification.

Puis cliquez sur **SUIVANT** pour voir le résumé de votre planification, ou **TERMINER** pour la valider.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-13/8d6a9290-2831-4817-9595-cadc67c35926/File_c0fedaa229194167b91ae0a107e7d7e2_text_export.jpeg)
