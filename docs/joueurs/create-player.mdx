# Créer un joueur sur MyCoach Pro.



1\. Cliquez sur le module "**Joueurs**".

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/5ce98701-15b6-4b99-9e5b-eefd8eb48470/ascreenshot_4da8bcb5cf3b40a796969f9fca2ae942_text_export.jpeg)


2\. Puis cliquez sur le bouton "**Ajouter un joueur**".

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/0a906ff7-0ea1-4418-950f-4acc4be13679/ascreenshot_10c8f7a0d79a40d1852c8b7361a3a696_text_export.jpeg)


3\. Remplissez les différents champs disponible, attention certains sont **obligatoires**, comme **l'identifiant**, le **mot de passe**, et le **poste du joueur.**\
\
**ATTENTION: un identifiant MyCoach Pro est unique, il ne peut être utilisé pour plusieurs comptes différents.**

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/9a9d04cb-ccad-4ec2-8a46-7decd1ccdc27/ascreenshot_b8390859b8d340f9994e4dff5ebede18_text_export.jpeg)


4\. 

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/bab38ca5-925f-40c6-9c52-6916dd1cb34e/ascreenshot_e0ad9e12054541ff9f20335db452cd6a_text_export.jpeg)


5\. **Attention**, les mots de passe sont standardisés selon les critères discriminants suivants:\
**\- 8 caractère minimum\
\- 1 minuscule\
\- 1 majuscule\
\- 1 chiffre\
\- 1 caractère spécial (@#%$&%\*!...)**

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/3d0fae40-97d8-4a1a-8e1c-6378dafe86a3/ascreenshot_8ccff6d06c744882be3dd3fa24c5bd2f_text_export.jpeg)


6\. 

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/276d6a6f-ff17-441e-bde9-0dbee6ef6038/ascreenshot_f713ff2b15a94d6093eee42c527868f8_text_export.jpeg)


7\. 

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/437a797f-a8d8-4252-b761-defa7a3735a2/ascreenshot_a52927d62ce94bef8abaff687a687057_text_export.jpeg)


8\. 

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/c753fbf4-004d-426e-98d9-aab4d885c646/ascreenshot_967f9db2bc6648a6a36b2d8bcbc102f2_text_export.jpeg)


9\. **IMPORTANT:** Sélectionnez les **modules auxquels vous souhaitez donner accès à vos joueurs**. Exemple ici, **le module séance joueur** leur permet de **créer des séance depuis l'application MyCoach Pro - AMS.**

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/7ed8b227-22d2-4218-aa39-bb611d92be76/user_cropped_screenshot_9f3f8e318b0f4fc5806b012f8bac1d55_text_export.jpeg)


10\. Une fois la fiche remplie à votre convenance, cliquez sur "**TERMINÉ**".\
Vous retrouverez votre joueur dans la liste de vos effectifs.

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/08e50256-b9ea-44e6-aab5-06e1541c5e89/ascreenshot_c00f8b865db14cf6bbd8dd83b5e72969_text_export.jpeg)
