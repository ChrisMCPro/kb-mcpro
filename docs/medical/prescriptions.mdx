# Créer une prescription médicale


1\. Dans le *panneau des modules*, cliquez sur le module **Médical**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/befc2e5e-cd80-40ba-90d8-e3048ff4c870/ascreenshot_9597286411fc40cabdb0f2e9b7107bc0_text_export.jpeg)


2\. Cliquez sur l'icone  **+**  à côté du joueur pour lequel vous souhaitez créer une prescription, pour ouvrir le sous menu.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/f7d42524-eb81-416c-9440-9374a19cf0f9/ascreenshot_0209cb6180944526aec279c4905b3c21_text_export.jpeg)


3\. Cliquez sur **Ajouter une prescription**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/c5ffb202-f890-471e-af47-ad0743ec23b4/ascreenshot_bc3ed2afd7ea4b458f41eb1bb6c0e4dd_text_export.jpeg)


4\. Choisissez la date de la prescription.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/ba3ecb98-cb0d-4889-aa7a-dd2a35b77fe4/ascreenshot_43e01410bfbe415191d91c7a7ae341b6_text_export.jpeg)


5\. Remplissez si besoin les champs de commentaires médical\* et partagé.

\
\**ce type de commentaire est seulement visible par les utilisateurs ayant le droit **medical** actif sur leur accès.*

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/46684865-ca95-4a7e-a58b-3a347db9fc99/ascreenshot_2a0f61b9246e4a75a8ffc0dc3fb3ffce_text_export.jpeg)


6\. Vous pouvez maintenant ajouter un médicament.

Pour cela deux options sont disponibles:

1. En le choisissant depuis une liste existante en cliquant dans le champ **Rechercher un médicament**.
2. En le créant en cliquant sur le bouton **+** qui ouvrira la fenêtre de création de médicament. (vous reportez à la section *Comment créer un médicament?*)

Vous avez également la possibilité de créer une prescription sans médicament en cliquant sur le bouton **AJOUTER UN TRAITEMENT SANS MÉDICAMENT.**

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/4bf02432-7ef4-4e0e-911c-5bfc906924cf/user_cropped_screenshot_9b0201464fbf4e09a71e342425300d8c_text_export.jpeg)


7\. Si vous choisissez l'option 1, sélectionnez 1 médicament dans votre liste.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/a26d0d6a-77bf-4382-8334-bdb085e4bd67/ascreenshot_72157bf4df924594b29e2417b53f2b56_text_export.jpeg)


8\. Une nouvelle fenêtre s'ouvre et vous permettra de configurer la prise de médicament de manière simple en choisissant une configuration pré-définie pour la posologie.\
\
Cependant vous pouvez tout de même la configurer manuellement comme dans l'exemple suivant.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/d22568b8-637b-4797-8e39-47f3a45d8281/ascreenshot_eff206c8b16146b7b4bf9b4cc81a2801_text_export.jpeg)


9\. Remplissez si besoin les champs de commentaires médical\* et partagé pour la prescription du médicament.

\
\**ce type de commentaire est seulement visible par les utilisateurs ayant le droit **medical** actif sur leur accès.*

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/29e856c0-51de-4ad9-9342-7f447437c527/ascreenshot_da671f9a7fc74c878e3137d0999cebd5_text_export.jpeg)


10\. Configurez ensuite le type de prise en choisissant parmi les options disponibles.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/41f7782a-da0c-4203-a0e2-d87e58dc41fc/ascreenshot_a5e5ffcfb2d84e6e9eca0ee4c5f79163_text_export.jpeg)


11\. Le dosage.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/b826728a-c18d-4f9b-bdc7-5633da03407e/ascreenshot_a79a2f65e975492d846c2170202a134d_text_export.jpeg)


12\. La fréquence.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/77b18ec3-6b40-4f17-aad0-044cf10b599c/ascreenshot_35d8c85844084e7cad11c20c5b9ee951_text_export.jpeg)


13\. 

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/09e9cf26-798c-4fe7-8647-dea258ba1677/ascreenshot_a08953d8318640d09790b0544cda6703_text_export.jpeg)


14\. La durée de la prise.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/20f1b23e-9b55-4150-b6ef-fe417de03b3f/ascreenshot_0b5a119e6d1e464a9dc119fcf169c831_text_export.jpeg)


15\. Configurez ensuite le dosage.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/ee8eb0b7-c403-4f2e-9ba5-0324634515b2/user_cropped_screenshot_24ea428ecca94c228ad01d5901912627_text_export.jpeg)


16\. Puis cliquez sur le bouton **SAUVER** pour terminer la configuration de ce médicament.\
\
Si vous devez ajouter un autre médicament **recommencez depuis l'étape 7**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/66079359-309b-4d9c-b1a6-cc15a88546a6/ascreenshot_95f971e9ff7f461da2e3549653dfaf0a_text_export.jpeg)


17\. Cliquez sur **TERMINER** pour finir la création de la prescription.

![](https://colony-recorder.s3.amazonaws.com/files/2025-02-10/098eb2bc-8f1c-448e-a074-fff8054d74a0/ascreenshot_a5d38c6ed029468b9152081311fa0576_text_export.jpeg)
