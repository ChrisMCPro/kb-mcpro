# Ajouter des chirurgies à un patient

Apprenez comment ajouter des chirurgies depuis notre application WEB sur vos patients.

Tip: Seule les personnes ayant le rôle Médical, et par conséquent la possibilité de créer / éditer des pathologies auront la possibilité d'ajouter des chirurgies sur les pathologies des patients auxquels ils ont accès.\
\
Les chirurgies ne peuvent être ajoutée que sur des pathologie de type [[Blessure]]  .


### Ajout d'une chirurgie à la création d'une pathologie


Depuis la création d'une pathologie, si vous cochez la case[[ intervention chirurgicale ]]dans la section Description de la blessure, de nouveaux champs sont disponibles pour ajouter des chirurgies.

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-05/84a53e61-d5d4-4f3d-924f-8894d4762e07/ascreenshot_957a9e613eb44563b4b42841a5c00930_text_export.jpeg)


Ajouter une chirurgie comprend:

- La date de la chirurgie
- Le chirurgien opérant
- Un champs commentaire
- Définir si elle est de [[référence]]ou non

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-05/a3194757-1cc5-48cd-8ac7-a839dfe412a5/File_41e678b2f0b04428afa5c7e002b38f05_text_export.jpeg)


Le bouton Ajouter une chirurgie vous permet d'ajouter autant de chirurgies que nécessaire.

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-05/b0fe2520-0391-4765-92eb-93ccca6b7bcc/ascreenshot_8bd9bbddf91948dfb07c14bbd34a4d81_text_export.jpeg)


Choisissez la date pour chaque chirurgie, elle seront classé par ordre chronologique dans le détail de la pathologie

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-05/5de49a81-3884-4ff3-991e-271712117248/ascreenshot_07fed1cdecd14eba99d75c61207ede19_text_export.jpeg)


Une seule chirurgie peut être la chirurgie de référence.\
Si une seule chirurgie est renseignée dans la pathllogie, elle sera la chirurgie de référence.

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-05/307254dc-873d-477e-b1aa-820e4a2920ff/ascreenshot_b7a7580e9ab5428aa6c60293dfd8131b_text_export.jpeg)


Ce bouton vous permet de supprimer une chirurgie si nécessaire.

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-05/02b71fa6-59e1-47b5-b483-a99fe702958c/ascreenshot_24d1ca673a9741379e3bdd9e4d0975d3_text_export.jpeg)


### Ajouter une chirurgie depuis la liste des pathologies d'un patient


Depuis la liste des pathologies, cliquez sur le bouton options.

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-05/b0e545df-c575-4baf-866e-d115c421bac8/ascreenshot_cc20373c91a2468ea7455cfc2e5f4ede_text_export.jpeg)


Sélectionnez "[[Ajouter une chirurgie]]" dans la liste des options.

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-05/60d4e883-daed-4d84-adf8-33d6835bed77/ascreenshot_a6ebb68e0f744985a9131b3d2c6595f7_text_export.jpeg)


Dans la modale, renseigner les informations de la chirurgie que vous souhaitez renseigner.

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-05/d39587fc-3d57-41d4-b194-01377a9529b2/stack_animation.webp)


Indiquez également si il s'agit de la chirurgie de référence.

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-05/fd581b6c-bea2-4f0f-a06f-205b3b54634b/ascreenshot_ce692cb177bd4de0ae7f362624117c92_text_export.jpeg)


Cliquez sur [[Enregistrer]] pour terminer.

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-05/81591d5c-ef5e-42b1-9916-818e5b6c4c2f/ascreenshot_2f6847df349a4c23b324d9ebc32b9ba2_text_export.jpeg)


### Ajouter une chirurgie depuis le détail d'une pathologie


Depuis le détail d'une pathologie, cliquer sur ce bouton vous permet d'ajouter une pathologie.

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-05/7736d483-fa87-48aa-a0bf-a1d94f37a37a/ascreenshot_99423ea30dc748df94e48c91594ae93c_text_export.jpeg)


Dans la modale, renseigner les informations de la chirurgie que vous souhaitez renseigner.

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-05/ca4e55c8-4df8-44d1-88ed-ab181bca2c18/stack_animation.webp)


Indiquez également si il s'agit de la chirurgie de référence.

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-05/ad6575fd-8dfc-4406-86ec-9e1b11e93065/ascreenshot_e3eb14945ed34d22a6df7592f477fdfe_text_export.jpeg)


Cliquez sur [[Enregistrer]] pour terminer.

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-05/cf7fd8ef-5663-4d21-bd17-a7190f83ac57/ascreenshot_161dde967b5d4a3e9d01723adf7979f2_text_export.jpeg)


 Dans le détail d'une pathologie, les chirurgies sont affichées dans l'ordre chronologique et une icone indique la chirurgie de référence.

![](https://colony-recorder.s3.amazonaws.com/files/2026-05-06/9c0a1db2-6dda-429c-a353-3cdd527d04e4/screenshot_701e877378154a0282c18fa705edc117_text_export.jpeg)
