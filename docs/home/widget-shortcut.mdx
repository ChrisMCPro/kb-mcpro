# Widget: Raccourcis


#### Le widget Raccourcis


1\. Le widget **Raccourcis** vous permet de créer différents raccourcis depuis votre accueil afin de faciliter votre navigation sur la plateforme ou d'épingler des actions que vous effectuez souvent. \
\
Vous avez la possibilité de l'ajouter plusieurs fois, et d'ajouter jusqu'à 10 raccourcis par widget!


2\. Depuis le module **ACCUEIL**, cliquez sur éditer l'accueil.

![](https://colony-recorder.s3.amazonaws.com/files/2025-04-25/86511eac-252d-48f7-af90-0579c2394566/ascreenshot_f8ad0637255f47c6a72b4beca8d39868_text_export.jpeg)


3\. Cliquez ensuite le bouton **+** pour afficher la liste des widgets.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-23/7944aa89-ba10-4326-8575-105e07cf5c44/user_cropped_screenshot_ab62472b06964fe48768f1266217208c_text_export.jpeg)


4\. Sélectionnez le widget **Raccourcis** pour commencer à le configurer et cliquez sur **SUIVANT**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-04-25/58ccee4c-4e27-453e-8e46-38be305295ff/ascreenshot_334326d59680461c8b91344134b96873_text_export.jpeg)


5\. 

![](https://colony-recorder.s3.amazonaws.com/files/2025-04-25/50fae6d5-6ac9-4631-a9de-a04aa8b6db54/ascreenshot_6d1bc05f12c5443ab1aa6522b5989865_text_export.jpeg)


6\. Configurez le widget avec 1 ou plusieurs raccourcis en éditant la couleur d'en-t6ete, le titre du widget et ceux des raccourcis que vous ajouterez (10 au maximum).

![](https://colony-recorder.s3.amazonaws.com/files/2025-04-25/1955ddc4-80d8-47d4-8241-1239c7681a97/ascreenshot_ea47f530486742228c148d5778e419b7_text_export.jpeg)


7\. **Les raccourcis peuvent rediriger** vers **un module en particulier**, comme le calendrier joueurs ou **une action spécifique** comme la création d'un événement.

![](https://colony-recorder.s3.amazonaws.com/files/2025-04-25/0accbd02-2e4e-4fae-bbe3-8a45eb76e829/ascreenshot_4150a1cf6ca0460c912cd76817dd4b50_text_export.jpeg)


8\. Cliquez sur **AJOUTER UN RACCOURCI** pour en ajouter d'autres puis configurer les.

![](https://colony-recorder.s3.amazonaws.com/files/2025-04-25/f0135ec9-c757-4065-8e68-7dbc03622721/ascreenshot_27d774cb8b514e9ea69d6c715d996156_text_export.jpeg)


9\. 

![](https://colony-recorder.s3.amazonaws.com/files/2025-04-25/6463936f-5ed8-4507-b132-f7ef00a9ce3d/ascreenshot_766cf11ae60a434ebfe919d7ca45ab80_text_export.jpeg)


10\. Une fois tous vos raccourcis configurés, cliquez sur **TERMINER**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-04-25/64106e30-a059-469d-a66f-36a41bcdfaa2/ascreenshot_6141a2e074e34db5b32911d202098be3_text_export.jpeg)


11\. Quittez le mode édition

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-23/0f8e3e09-961e-445c-bfa5-28c87f6ebbdb/user_cropped_screenshot_9657decbebeb4f16a779a042e57cef40_text_export.jpeg)


12\. ***En cliquant sur le raccourci...***

![](https://colony-recorder.s3.amazonaws.com/files/2025-04-25/1659e4a3-e4ac-41cd-ab7f-7d5742a84b62/ascreenshot_f93341039b8b4ed987443d3134e946d4_text_export.jpeg)


13\. ***j'arrive directement dans la planification de séances d'entraînement.***

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-02/a8cff5eb-aa24-4512-976b-09fe3d1bcd65/screenshot_e7ddeac516034946aa824df655499b9c_text_export.jpeg)
