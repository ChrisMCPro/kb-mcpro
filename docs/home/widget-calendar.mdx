# Widget: Calendrier


1\. Le widget calendrier permet un coup d'œil rapide sur le planning des **7\*** prochains jours depuis l'**Accueil**, à partir de la date du jour.\
\
\*7 étant le nombre maximum de jours affichables.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-23/49ba59a5-73f7-4831-989f-4b90251cb37e/screenshot_0607df427fca48228d71087d41e7c50b_text_export.jpeg)


2\. En cliquant sur le bouton **Éditer l'accueil**, puis sur le bouton de configuration du widget **Calendrier**

![](https://colony-recorder.s3.amazonaws.com/files/2025-04-25/ef7798dd-2f2a-4dc5-abf6-b505793be0de/ascreenshot_9ce97b65ae1e4a1f9c7695280d0a0166_text_export.jpeg)


3\. Click this icon.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-23/5e9a6342-3a88-4155-8358-dbfc72547a4a/user_cropped_screenshot_bd68ded7fbba451e83ddc9214ca31cc9_text_export.jpeg)


4\. Vous avez la possibilité de modifier le nombre de jours affichés dans le widget, **entre 1 et 7**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-23/e371d2dc-b9d4-4835-8c00-4f36a712d29b/screenshot_222cf82008ce47e5a8cb6a60cffe0138_text_export.jpeg)


5\. Cliquez sur **TERMINER** lorsque vous avez effectué la modification souhaitée.

![](https://colony-recorder.s3.amazonaws.com/files/2025-04-25/78e8a72f-68fc-49de-90e3-edae07e19099/ascreenshot_d9c8714d624b431fb91e8e8575bb06b2_text_export.jpeg)


6\. Cliquer sur une date affiche toutes les séances prévues à cette date.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-23/41c2b499-900f-4c18-a47c-44970cc4cee8/user_cropped_screenshot_aee8a0324d8c4e88a67011ad2f124b56_text_export.jpeg)
