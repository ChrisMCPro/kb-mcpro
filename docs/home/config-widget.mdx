# Configurer ses widgets


1\. L'accueil est votre premier contact avec MyCoach Pro lors de votre connexion.\
Nous l'avons donc rendu personnalisable par utilisateur afin qu'ils corresponde au besoin de chacun.

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/139e1905-cba5-4036-876d-4d92a145d6cf/screenshot_189a68f53db44e01ac851d7948f1fda9_text_export.jpeg)


2\. Pour l'éditer, commencez par cliquer sur le bouton "**Éditer l'accueil**".

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/25c5043c-ab21-4321-9314-d1bec37d6064/ascreenshot_df6189e061d1478f8c66818ea3c17ca4_text_export.jpeg)


3\. Vous trouverez tout **en bas de la page un bouton** "**\+**", c'est ce dernier qui vous permettra d'**ajouter des widgets** parmi la liste disponible (elle varie selon vos droit et votre niveau d'accès).

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/12cbab7b-d04f-4e76-bc3b-f40eef6618ba/ascreenshot_fd080b542db24794b27a0e7df812263c_text_export.jpeg)


4\. Exemple d'ajout d'un widget ***États de forme*** (nécessite les accès au module médical).

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/c2e0a411-1833-4c41-9f84-b1de34c12723/ascreenshot_431458243d0b4f7b857ddd8454884b18_text_export.jpeg)


5\. Une fois votre widget sélectionné, cliquez sur **SUIVANT**.\
Donnez un titre à votre widget

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/94a5d9b3-34c2-46e7-8ce0-6a362dcca03d/ascreenshot_6ea9562070c143de97481d047efe7aa5_text_export.jpeg)


6\. 

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/01f8f11d-bf3d-443a-929e-83f0c7a23bb9/ascreenshot_4ae24ae2388349779d84c85af33e2cce_text_export.jpeg)


7\. Sélectionnez les états de forme que vous souhaitez voir sur ce widget.

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/489c47e1-4e1f-4103-b5a9-e0b74b06a424/ascreenshot_a3ffdb44ca834ff694a059e3fef62fc7_text_export.jpeg)


8\. Choisissez la **taille du widget et sa couleur** afin de mieux l'identifier (exemple rouge pour un widget concernant les joueurs blessés).\
\
Une fois votre widget paramétré, cliquez sur **TERMINÉ.**

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/56d94ccd-1eb5-4109-94a0-b0433e8d91ca/ascreenshot_10af9bb065994686a7fe77b00b32331e_text_export.jpeg)


9\. 

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/50a101dd-e3a0-49ef-95fd-50bc983d3643/ascreenshot_b6af45c09ce244a5a446e89c3173cb2f_text_export.jpeg)


10\. 

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/f2232d51-8901-4467-8f28-46c8776832a1/ascreenshot_21d0d8153b4a495aacdafcbcc8ebdd66_text_export.jpeg)


11\. Vous avez la **possibilité de supprimer les widgets** dont vous n'avez pas besoin, afin de **ne pas "polluer" votre accueil avec des infos ou des widgets superflus.**

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/6e3be03b-ca7b-445f-b764-73479eec5de3/ascreenshot_45a07f4c3dd140a8bcbc185ab50bc5cd_text_export.jpeg)


12\. 

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/c0aa12b1-93e8-4e8f-b996-2cb0e98828c6/ascreenshot_e8d2c7055a4b4251b80bb316eab5b2c2_text_export.jpeg)


13\. Pour **quitter le mode édition**, cliquez sur le bouton en haut à droite "**Quitter le mode édition**".

![](https://colony-recorder.s3.amazonaws.com/files/2023-10-09/0f126f94-4132-4098-ad48-8863aae806f1/ascreenshot_8044bd2bda8c42bc9f65e4a6a14eeb21_text_export.jpeg)
