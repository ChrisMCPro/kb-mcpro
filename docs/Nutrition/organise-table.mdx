# Ordonner les colonnes du tableau de données Nutrition.
#### [Made by Support Team MyCoach Pro with Scribe](https://scribehow.com/shared/Ordonner_les_colonnes_du_tableau_de_donnes_Nutrition__KSE52KTOSS24pafk7AFkZQ)


1\. Pour modifier l'ordre des colonnes du tableau de suivi athlétique, il faut tout d'abord se rendre dans le module **Nutrition**

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-26/131d62da-7f56-4710-9aad-5110e0872d0a/user_cropped_screenshot_387257a0ea4744d787f2324396506552_text_export.jpeg)


2\. En **haut à droite** cliquez sur le **bouton des filtres**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-26/fceda471-b6b1-48a2-981b-9452464cf99c/user_cropped_screenshot_405300b54bed4862b2ea68cbb52603d9_text_export.jpeg)


3\. Depuis ce panneau latéral, **vous pouvez modifier l'ordre** des colonnes **en cliquant sur les flèches haut et bas**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-05-26/c0da4416-3186-4213-8fef-801f9bca8621/user_cropped_screenshot_fbf0f8da199048f1947c86bc30ea8053_text_export.jpeg)


Tip: **Les modifications seront enregistrée dans les préférences utilisateur.**\
**Chaque utilisateur peut donc avoir sa propre visualisation**.
