# Mes joueurs ne reçoivent plus de notifications!
#### [Made by Support Team MyCoach Pro with Scribe](https://scribehow.com/shared/Mes_joueurs_ne_reoivent_plus_de_notifications__SsNpoowlRh-0To37Y0iMkA)


#### Identifier le problème


1\. **1- Qui? Depuis quand?**

\
\
L'élément le plus important est de **bien identifier qui parmi votre effectif** a réellement un problème pour recevoir les notifications MyCoach Pro.

- Qui?
- Depuis combien de temps?
- Cela se produit-il sur tous les événements ou il a été convoqué ou sur un type d'événement en particulier?


2\. **2- Vérifier que mon joueur a bien activer les notifications.**\
\
\
Le cas le plus probable est que l'utilisateur concerné n'ai pas activer ou bien refuser les notifications sur son appareil mobile.\
\
Vous trouverez en détail ici comment y remédier en fonction du type d'appareil.


#### Comment vérifier et autoriser les notifications sur mon appareil?


3\. ***iOS (iPhone)***

1. Aller dans l’application **Paramètres.**
2. Rendez-vous dans la section **Apps\*** (dernière mise à jour) ou trouver le logo de **MyCoach Pro - AMS** et appuyez dessus.
3. \*Après avoir appuyé sur **Apps**, trouvez le logo de **MyCoach Pro - AMS** et appuyez dessus.
4. Dans cette nouvelle section appuyer sur **Notifications**
5. Tout en haut vous verrez un bouton **Autoriser les notifications.**\
   Si ce bouton est gris, alors il faut appuyer dessus pour activer les notifications.


4\. ***Android (l'expérience peut varier d'un modèle à l'autre, nous proposons ici la solution la plus commune).***

1. Trouver l'application **MyCoach Pro - AMS** et appuyer dessus en maintenant l'appui pour afficher le menu contextuel.

2. Il faut ensuite appuyer sur le bouton **Infos sur l’appli**.

3. Dans le nouveau menu, appuyer sur la section **Notifications**.

4. Un encadré avec un bouton nommé **Toutes les notifications de MyCoach Pro - AMS** apparaît. \
   Appuyez dessus pour l'activer si ce n'est pas fait:\
   \
   \
   Si le bouton n'était pas actif, et que vous venez de l'activer, assurez-vous de **fermer l'application** ([Comment fermer une application sur un appareil Android?](https://support.google.com/android/answer/9079646?hl=fr#zippy=%2Cfermer-des-applications)).

   \
   \
   Une fois l'application relancée, **elle vous proposera d'autoriser les notifications** comme lorsqu'elle a été lancée la première fois. **Appuyez sur Autoriser**.


#### Cela ne fonctionne pas ou je ne suis pas dans le cas décrit plus haut.


5\. Si ces manipulations ne fonctionnent pas, ou que vous n'êtes pas dans ce cas, et que vos notifications étaient bien autorisées pour l'application **MyCoach Pro - AMS** sur votre appareil, veuillez essayer cette dernière manipulation:

1. Lancez l'app **MyCoach Pro - AMS** et appuyez sur le bouton **Menu**
2. Appuyer ensuite sur la section **Diagnostic**.
3. Appuyez sur le bouton **Resynchroniser le token de notification**.

Dans le cas ou elle ne fonctionnerait pas veuillez nous contacter à [support@mycoachpro.io](mailto:support@mycoachpro.io) en nous fournissant un maximum de détail tels que:

- Depuis quand vous rencontrez le problème, plus ils nous est remonté tôt, plus il sera simple de le résoudre.
- Les utilisateurs concernés et l'appareil qu'ils utilisent.
- Les événements concernés (type, date, heure)

Ces informations nous permettront d'agir plus rapidement et efficacement.
