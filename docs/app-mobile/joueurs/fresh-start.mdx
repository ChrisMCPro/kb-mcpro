# Démarrer avec l'application mobile

:::info Pour qui ?
Ce guide est destiné aux **joueurs** qui se connectent pour la première fois à l'application MyCoach Pro AMS.
:::

---

## Étape 1 — Activez votre compte

Vous recevez un email de bienvenue contenant :

- **Votre nom d'utilisateur** (il peut différer de votre adresse email)
- **Un lien pour créer votre mot de passe** (valide 24h)

Cliquez sur le lien pour accéder à la page de création de mot de passe.

![Email de bienvenue MyCoach Pro](https://colony-recorder.s3.amazonaws.com/files/2024-07-18/792cf9e5-7392-491d-b5fb-0c19be3a3fd4/user_cropped_screenshot_2f851d8ec05d4914ac233aa74ef1df65_text_export.jpeg)

---

## Étape 2 — Créez votre mot de passe

Sur la page qui s'ouvre :

1. Saisissez votre mot de passe (respectez les critères indiqués)
2. Confirmez votre mot de passe
3. Cliquez sur **ENVOYER**

![Page de création de mot de passe](https://colony-recorder.s3.amazonaws.com/files/2024-07-18/73ff59bf-2c75-45ef-bf65-977e67c4ca01/user_cropped_screenshot_91f1005406d4478eb9e31ce952213719_text_export.jpeg)

---

## Étape 3 — Confirmation

Votre mot de passe est créé. Vous pouvez maintenant télécharger l'application.

![Confirmation de création de compte](https://colony-recorder.s3.amazonaws.com/files/2024-08-09/1b22b161-d05c-40f1-b3c0-75e7b094e6af/user_cropped_screenshot_d32d1c2040c740c4ab29cf171df5c167_text_export.jpeg)

---

## Étape 4 — Téléchargez l'application

L'application **MyCoach Pro AMS** est disponible sur :

- 📱 [App Store (iOS)](https://apps.apple.com/fr/app/mycoach-pro-ams/id1643524841)
- 🤖 [Google Play (Android)](https://play.google.com/store/apps/details?id=com.mycoachpro.ams)

![Stores iOS et Android](https://colony-recorder.s3.amazonaws.com/files/2024-07-18/eeddbbc5-72a0-4e53-95d3-61dd96c82183/user_cropped_screenshot_849c4fc6698047a794230ef4531a4ed5_text_export.jpeg)

---

## Étape 5 — Connectez-vous

Une fois l'application installée :

1. Ouvrez **MyCoach Pro AMS**
2. Saisissez vos identifiants (nom d'utilisateur + mot de passe)
3. Appuyez sur **Connexion**

![Écran de connexion](https://colony-recorder.s3.amazonaws.com/files/2024-07-18/3ba3d4e5-47e9-4b3c-8789-1023804e5a54/mobile_screenshot_ee41fb87de4940c1aaf857322fa196cf_text_export.jpeg)

---

## Étape 6 — Autorisez les notifications

Pour ne manquer aucune convocation ou demande de formulaire, appuyez sur **Autoriser** lorsque l'application vous le demande.

![Demande d'autorisation des notifications](https://colony-recorder.s3.amazonaws.com/files/2024-07-18/6353c225-1abb-4849-a318-16844c02cbfc/mobile_screenshot_742ac32f890b4b38be480d805e949c51_text_export.jpeg)

:::tip Problème de connexion ?
Vérifiez que votre **nom d'utilisateur** est correct — il peut différer de votre adresse email. En cas de doute, contactez votre staff.
:::
