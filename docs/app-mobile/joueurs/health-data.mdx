⬇️ Importer son activité en tant que séance joueur

Une fois vos activités synchronisées dans MyCoach Pro AMS, vous avez la possibilité de les importer en tant que séances joueur.

* Depuis l’Accueil ou le module Données Santé, sélectionner l’activité à importer.
* Depuis l’activité sélectionnée, cliquez sur Créer une séance joueur.
* Configurer votre séance, elle sera créée en intégrant les données de votre activité.

F.A.Q.

Je ne trouve pas certaines de mes activités Polar Flow ou Garmin Connect dans MyCoach Pro AMS, que dois-je faire ?
Si certaines de vos activités Polar ou Garmin n’apparaissent pas dans MyCoach Pro AMS, voici comment résoudre le problème :
      1. Vérifiez que l’activité existe dans l’application d’origine
Assurez-vous que l’activité est visible dans Polar Flow ou Garmin Connect.
      2. Vérifiez que l’activité est bien envoyée vers Apple Health / Health Connect
        1. Appareils Android
          1. Ouvrez Paramètres → Sécurité et confidentialité → Health Connect → Données et accès.
          2. Vérifiez si des données (distance, fréquence cardiaque, etc.) sont enregistrées aux dates concernées.
        2. Appareils Apple
          1. Ouvrez l’application Apple Health puis allez dans l’onglet Parcourir.
          2. Sélectionnez une catégorie (ex. : Activité, Cœur) et vérifiez si des données sont enregistrées aux dates concernées.

        Si aucune donnée n’est disponible, il est normal que rien n’apparaisse dans MyCoach Pro AMS.

      3. Forcer une nouvelle synchronisation depuis Polar Flow ou Garmin Connect
Vous pouvez relancer la synchronisation de vos activités depuis l’application Polar Flow ou Garmin Connect.
Cela n’affectera pas les séances déjà importées dans MyCoach Pro AMS.

Si le problème persiste, n’hésitez pas à contacter le support en fournissant :
        * La date de l’activité manquante et une capture d’écran depuis Polar / Garmin
        * Une capture d’écran de votre Health Connect
        * Des captures d’écran de votre application Apple Health / Health Connect
Comment fonctionne le processus de synchronisation des données de santé ?
  * La synchronisation des données de santé suit plusieurs étapes afin de permettre la synchronisation de vos activités dans l’application mobile MyCoach Pro AMS.
  * Si l’une de ces étapes de synchronisation ne fonctionne pas correctement, vous ne pourrez pas voir vos activités dans MyCoach Pro AMS.
  * Comme mentionné dans le sujet ci-dessus, le problème le plus fréquemment signalé concerne principalement une synchronisation défaillante entre votre application d’activité (Polar Flow / Garmin Connect / Whoop etc.) et l’application santé de votre appareil (Health Connect / Apple Health).
